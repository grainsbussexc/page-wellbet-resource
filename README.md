# page-wellbet-resource

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态资源的形式存放，不针对任何特定域名或网站，仅作为通用页面存储与分发用途。

仓库内的页面均为独立编写，彼此无依赖关系，可直接在浏览器中打开运行。

## 目录说明

```
page-wellbet-resource/
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 公共资源文件（如 CSS、JS、图片等）
├── docs/           # 文档或说明文件（可选）
└── README.md       # 本文件
```

- `pages/`：每个 HTML 文件为一个独立页面，文件名通常反映其功能或用途。
- `assets/`：存放页面可能引用的样式表、脚本或图像。如无引用，此目录可为空。
- `docs/`：用于存放额外的使用说明或设计文档。

## 页面归档说明

- 所有页面均为静态 HTML，不依赖后端服务。
- 页面之间互不关联，可单独部署或访问。
- 页面内容不包含任何商业推广、敏感信息或指向具体网站的链接。
- 归档的页面可能包含演示、工具、展示或测试用途的界面。

## 维护说明

- 本仓库由维护者不定期更新，新增或修改页面。
- 如需提交页面，请将 HTML 文件放入 `pages/` 目录，并确保所有引用资源均位于 `assets/` 内。
- 不鼓励在页面中加入外部依赖（除非必要），以保证离线可用性。
- 若发现页面无法正常显示或存在错误，欢迎提交 Issue 或 Pull Request。

## 使用方式

1. 克隆本仓库到本地：
   ```
   git clone https://github.com/your-username/page-wellbet-resource.git
   ```
2. 进入 `pages/` 目录，直接双击 HTML 文件即可在浏览器中查看。
3. 也可将 `pages/` 目录托管至任意静态服务器（如 GitHub Pages、Netlify 等）以在线访问。
